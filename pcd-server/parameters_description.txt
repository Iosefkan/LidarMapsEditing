Параметры 2.5D-сетки и кандидатов

--grid — размер клетки сетки (м).

--q_low — нижний квантиль Z в клетке (оценка «земли»).

--q_high — верхний квантиль Z в клетке (оценка «верха»).

--smooth_cells — сглаживание «земли» по соседям (радиус в клетках).

--h_min — минимальный «выступ» dh = z_high − z_ground (м), чтобы клетка стала кандидатом.

--h_max — максимальный допустимый «выступ» (м).

--density_min — минимальное число точек в клетке, иначе клетка игнорируется.

Параметры фильтра формы (по компонентам)

--min_len — минимальная длина компонента (м) вдоль главной оси.

--min_width — минимальная ширина компонента (м).

--max_width — максимальная ширина компонента (м).

--min_elong — минимальная вытянутость (длина/ширина).

Поиск длинных полос (Хафф) — опционально

--use_hough — применить трансформации Хаффа.

--hough_theta_step — шаг угла в аккумуляторе (°).

--hough_rho_bin — шаг по смещению линии ρ (м).

--hough_topk — сколько сильнейших линий брать для проверки.

--hough_min_len — минимальная длина полосы (м), чтобы принять её.

--hough_min_w — нижний порог фактической ширины полосы (м).

--hough_max_w — верхний порог фактической ширины полосы (м).

--hough_dilate — «утолщение» найденной полосы на r клеток (склейка разрывов).
